help_center Uso rápido
  • Usa la cinta superior para abrir Archivo, Manuales, Opciones y Ayuda.
  • Las tabs dinámicas se abren una sola vez; si ya existen, la app te lleva a ellas.
  • Pulsa Esc para cerrar la tab activa cuando sea cerrable.
  • La navegación intenta devolverte al tab desde el que abriste el contenido actual.
  • Perfil, Admin panel y las nuevas tabs de Ayuda no se abren por defecto.
 info_outline Acerca del proyecto

Fichas 3.5 es una aplicación web orientada a consulta, creación y gestión de fichas y catálogos de D&D 3.5.

El proyecto sigue en evolución y combina catálogo, navegación por pestañas, perfil de usuario y herramientas de administración para mantener los datos sincronizados.

Autor y desarrollo: yosiftware.es
